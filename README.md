# AimLock
A rookie made Aimbot, use at your own risk
## What is AimLock?
        
AimLock is code for a universal first person Roblox Lua Aimbot, ESP, Fly and Noclip cheat.

### Some Features
- Aimbot
  - Fov size, transparency
  - Smoothening
  - Team Check, Wall Check
  - Random Offset

- Esp
  - Name Esp
  - Distance Esp
  - Health Esp
- Fly
- Noclip
- Speed Hack
